Enum piston::event::Event[src]

pub enum Event {
    NoEvent,
    KeyReleased(Key),
    KeyPressed(Key),
    MouseButtonPressed(Button),
    MouseButtonReleased(Button),
    MouseMoved(f64, f64, Option<(f64, f64)>),
}

Contains the information associated with an event.

Variants

NoEvent

No event occured.

KeyReleased

A key was released.

KeyPressed

A key was pressed.

MouseButtonPressed

A mouse button was pressed.

MouseButtonReleased

A mouse button was released.

MouseMoved

The mouse moved.

(x, y, Some((dx, dy))) The second part is relative motion which is not bounded. Relative motion might not be supported by all window back-ends.
