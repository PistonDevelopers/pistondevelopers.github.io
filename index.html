Piston is a user friendly open source game engine written in Rust, started by Sven Nilsen.

The bricks are out of the box! Take a bite of some nice modular libraries for the real world, such as games and interactive applications, written for performance, ergonomics and cross platform development.

Piston is made by some very smart, awesome and well connected people. We work in a flat structure with no leader, giving maintainers more responsibility and power to direct projects. Everybody has full write access, we share research and we are part of a greater community. Our goal is not to make a "game engine". We have different personal goals, but share the cost of maintaining important libraries together. We care about every aspect of development, which means designing for the programmer, to make it possible to reach our goals. You can be a part of it.

Quick example
use event::{ EventIterator, EventSettings, Window, WindowSettings };
use gfx::{Device, DeviceHelper }; use gfx_graphics::{ G2D, Texture };
fn main() {
  let mut window = sdl2_game_window::WindowSDL2::new(
    shader_version::opengl::OpenGL_3_2, WindowSettings {
      title: "Hello Piston".to_string(), 
      size: [300, 300], fullscreen: false, exit_on_esc: true,
      samples: 0 });
  let mut device = gfx::GlDevice::new(|s| unsafe {
    std::mem::transmute(sdl2::video::gl_get_proc_address(s)) });
  let (w, h) = window.get_size();
  let frame = gfx::Frame::new(w as u16, h as u16);
  let mut renderer = device.create_renderer();
  let mut g2d = G2D::new(&mut device);
  for e in EventIterator::new(&mut window, &EventSettings {
    updates_per_second: 120, max_frames_per_second: 60
  }) {
    use event::RenderEvent;
    e.render(|_| {
      g2d.draw(&mut renderer, &frame, |c, g| {
        use graphics::*;
        c.rgb(1.0, 1.0, 1.0).draw(g);
        c.rect(0.0, 0.0, 100.0, 100.0).rgb(1.0, 0.0, 0.0).draw(g);
      });
      device.submit(renderer.as_buffer()); renderer.reset();
    });
  }
}
