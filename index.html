Piston

A user friendly game engine written in Rust

About Piston

Piston is a user friendly open source game engine written in Rust, started by Sven Nilsen.

Rectangle Example

{% highlight Rust %} use graphics::*; use piston::{Game, Gl, RenderArgs}; pub struct App { ... } // your application data impl Game for App { fn render(&self, c: &Context, args: RenderArgs) { c.rect(0.0, 0.0, 50.0, 50.0).rgb(1.0, 0.0, 0.0).fill(args.gl); } } {% endhighlight %}
